## Главное правило

Текст кодексов в `*/sources/` и полные файлы (`tax/kg-tax-code-2026-01-27.md`, `labor/kg-labor-code-full.md`) — **immutable, verbatim**.
**НЕ редактировать, НЕ перефразировать, НЕ «улучшать» формулировки.** Цитировать только дословно.
Менять можно только навигационный слой (`*/wiki/`, этот файл, README).

## Структура (одинаковая для обоих кодексов)

```
<кодекс>/
├── kg-*-full.md / kg-tax-code-2026-01-27.md   # весь кодекс одним файлом (греп по всему тексту)
├── sources/                                   # тот же текст по разделам (immutable) + оригинал (PDF/DOCX)
│   └── kg-*-code-rNN.md
└── wiki/
    ├── kg-*-code.md                           # хаб: разделы, главы, правила цитирования, оговорки
    └── kg-*-code-index.md                     # карта всех статей: № → заголовок → раздел/глава → файл
```

## Как отвечать на вопрос по кодексу (алгоритм)

1. Определи кодекс → открой `<кодекс>/wiki/kg-*-code-index.md`, найди статью по номеру или заголовку.
2. Колонка «Файл» → открой `<кодекс>/sources/kg-*-code-rNN.md`.
3. Найди точную норму грепом:
   - Налоговый: `^Статья N\.` (обычный текст)
   - Трудовой: `^\*\*Статья N\.` (заголовки жирным `**…**`)
4. Цитируй дословно, ссылайся на **номер статьи** (не на страницу источника).
5. Если нормы в тексте нет — скажи прямо «в данной редакции нет», не додумывай.

**Трудовой (`labor/`):**
- Статьи 1–265 без пропусков и вставных.
- Изменения Закона КР от 14.11.2025 № 257 вступают в силу с 01.01.2027 — в тексте есть пометки «Внимание!», они часть официального текста.

## Актуальность

Каждый кодекс актуален только на дату своей редакции (см. таблицу выше и frontmatter wiki-хабов). Изменения после этих дат здесь не отражены.
