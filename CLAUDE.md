# Налоговый кодекс Кыргызской Республики — правила для LLM

Это **дословный** текст Налогового кодекса КР + навигационный слой для ИИ.
Редакция: по состоянию на Закон КР от 31 декабря 2025 года N 262 (PDF от 2026-01-27).

## Главное правило

Текст кодекса в `sources/` и `kg-tax-code-2026-01-27.md` — **immutable, verbatim**.
**НЕ редактировать, НЕ перефразировать, НЕ «улучшать» формулировки.** Цитировать только дословно.
Менять можно только навигационный слой (`wiki/`, этот файл, README).

## Как отвечать на вопрос по кодексу (алгоритм)

1. Открой `wiki/kg-tax-code-index.md`, найди статью по номеру или по заголовку.
2. Колонка «Файл» → открой `sources/kg-tax-code-rNN.md`.
3. Найди точную норму грепом по началу строки: `^Статья N\.` (вставные: `^Статья N-M\.`).
4. Цитируй дословно, ссылайся на **номер статьи** (не на страницу PDF — страницы нестабильны).
5. Если нормы в тексте нет — скажи прямо «в данной редакции нет», не додумывай.

## Оговорки (важно)

- **Вставные статьи** вида `N-M` (34-1, 116-2, 249-2 … всего 24) — самостоятельные статьи, не части статьи N.
- **Статьи 403–408 отсутствуют** в данной редакции (изъяты) — пропуск нумерации не ошибка.
- **Глава 21 «Налоговый учет»** — вводная глава Особенной части, физически лежит в `sources/kg-tax-code-r07.md`.
- Текст извлечён из PDF (PyMuPDF, sort=true); убраны только колонтитулы и номера страниц, схлопнуты растянутые пробелы юстировки. Переносы строк внутри абзацев сохранены как в PDF.
- Актуальность ограничена редакцией 2026-01-27: изменения кодекса после этой даты здесь не отражены.
